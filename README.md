# Smart City Web Application

## 📌 Overview 
A full-stack Smart City web application developed using Java (JSP/Servlets) and MySQL.

The system provides a centralized platform where users can explore city services including tourism, education, businesses, and job opportunities, while administrators manage all system content.

## 🚀 Features

### 👤 User Features
- Browse tourism, jobs, businesses, and education services
- Search and filter city data
- User registration & login system
- Interactive map with city locations
- Password reset functionality

### 🔐 Admin Features
- User management (approve, reject, delete)
- CRUD operations for:
  - Businesses
  - Jobs
  - Tourism
  - Education
- Dashboard with system insights

## 🛠️ Technologies
- Java (JSP / Servlets)
- MySQL
- HTML, CSS, JavaScript
- Bootstrap
- Leaflet.js (Map integration)
